注意事项：
	1.在定时器3中断函数中，有个按键检测，模拟有车来，当按键按下时，RFLRState=RFLR_STATE_TX_START（最关键的一步），
只有进行了这一步，才能开始发送数据。还有就是千万不要改定时器3的时钟周期，因为SX1278官网驱动代码中，是以定时器3为
时钟周期，我怕改了，会出现一些bug。
    2.我把官方驱动代码整合成两个经常用的函数，即一个是发送函数，一个是接收函数，如果只是应用，只需要了解modbus.c
和modbus.h这个文件即可。如果你还想对一些配置参数进行调节，可以去sx1276-LoRa.c 129行----152行之间的代码修改。
